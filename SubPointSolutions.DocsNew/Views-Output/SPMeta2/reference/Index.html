Reference

SPMeta2 API reference

SPMeta2 introduces a domain of c# POCO objects, then it maps every single POCO object on SharePoint artifacts. There are a few important concepts in SPMeta2:

  • Definitions concept
  • Models concept
  • Provisioning services
  • Useful utilities

The following articles provide more information on how all that fits together, how the provision workflow looks like along with some usefule utilities you can also reuse in your projects.
